## 课程安排

操作系统：```Ubuntu 20.04```

编译器：```gcc/g++```

IDE：```Visual Studio Code/Visual Studio```

| 阶段 | 内容 | 目标 | 案例 |
|:----:|:----:|:----:|:----:|
| 第一阶段 | C++ 基础语法 | 介绍 C++ 基础语法知识，能有基础编程能力 | 通讯录管理系统 |
| 第二阶段 | C++ 核心编程 | 介绍 C++ 面向对象编程，为大型项目做铺垫 | 职工管理系统 |
| 第三阶段 | C++ 提高编程 | 介绍 C++ 泛型编程思想，及 STL 的基本使用 | 演讲比赛系统 |

最后综合大案例：机房预约系统
